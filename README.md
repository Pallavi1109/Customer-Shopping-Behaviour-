
# 🛒 Customer Shopping Behavior Analysis

An end-to-end data analytics project focused on understanding customer purchasing behavior using **Python, SQL, and Power BI**. The project explores shopping patterns, customer demographics, spending habits, and product preferences to generate actionable business insights.

---

## 📌 Project Overview

Businesses rely on customer behavior analysis to improve decision-making, optimize marketing strategies, and enhance customer experiences. This project analyzes customer shopping data to uncover trends and patterns that can help drive business growth.

The workflow includes:

* Data Cleaning and Preparation
* Exploratory Data Analysis (EDA)
* SQL-Based Business Analysis
* KPI Generation
* Interactive Dashboard Development in Power BI

## 🔍 Analysis Performed

### Data Cleaning

* Checked missing values
* Removed duplicates
* Standardized data formats
* Validated data quality

### Exploratory Data Analysis

* Customer demographics analysis
* Spending pattern analysis
* Product category performance
* Gender-based purchasing trends
* Age-group segmentation

### SQL Business Analysis

Key business questions answered:

* What is the total revenue generated?
* Which product categories perform best?

## 📈 Key Insights

* Customer spending patterns vary significantly across demographics.
* Certain product categories contribute a major share of total revenue.
* Repeat customers tend to generate higher average revenue.
* Purchase frequency has a strong impact on overall sales performance.
* Customer segmentation can help businesses target marketing efforts more effectively.
